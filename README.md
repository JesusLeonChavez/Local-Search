# Local Search

## Description
I am using a fictitious data, with the purpose of creating an algorithm that allows a search using react js

## STACK
- HTML5
- CSS3
- JavaScript
- ReactJs

## Installation

Clone the project

```bash
  git clone https://github.com/JesusLeonChavez/Local-Search.git
```

Installation

```bash
  npm i
```

Start Client

```bash
  npm start
```
